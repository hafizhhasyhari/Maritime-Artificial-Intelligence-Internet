* **Satu Data Indonesia:** [data.go.id](https://data.go.id/home)
    * Portal data nasional. Gunakan pencarian untuk "kelautan", "perikanan".
* **KKP (Kementerian Kelautan & Perikanan):** [satudata.kkp.go.id](https://satudata.kkp.go.id/)
    * Sumber data statistik perikanan, produksi, konservasi.
* **BRIN (Badan Riset & Inovasi Nasional):** [data.brin.go.id](https://data.brin.go.id/)
    * Mewarisi data dari LIPI (Oseanografi). Cari data ekspedisi, oseanografi.
* **BIG (Badan Informasi Geospasial):** [tanahair.indonesia.go.id](https://tanahair.indonesia.go.id/portal-web/)
    * Sumber utama data spasial (GIS) resmi Indonesia. Cari "Garis Pantai", "Batimetri", "Terumbu Karang", "Mangrove".
* **BMKG (Badan Meteorologi, Klimatologi, dan Geofisika):** [dataonline.bmkg.go.id](https://dataonline.bmkg.go.id/)
    * Sumber data iklim, cuaca, dan gempa.

## ğŸŒ Portal Data Global (Resolusi Tinggi)

* **Kaggle:** [kaggle.com/datasets](https://www.kaggle.com/datasets)
    * Dataset paling beragam dan bersih. Bagus untuk latihan model. Cari "Coral Bleaching", "Fish Species", "Ocean Data".
* **NOAA (National Oceanic and Atmospheric Adm.):** [data.noaa.gov](https://data.noaa.gov/datasetsearch)
    * Data oseanografi dan cuaca global (termasuk Indonesia). Sumber data Suhu Permukaan Laut (SST) terbaik.
* **Copernicus (Uni Eropa):** [marine.copernicus.eu](https://marine.copernicus.eu/)
    * Data observasi laut satelit (SST, klorofil, salinitas) dengan resolusi tinggi.
* **Global Fishing Watch:** [globalfishingwatch.org/data/](https://globalfishingwatch.org/data/)
    * Data pelacakan kapal (AIS) untuk memantau aktivitas penangkapan ikan.
