## ğŸš€ Langkah-Langkah Berkontribusi

Cara standar untuk berkontribusi melalui GitHub adalah sebagai berikut:

1.  **Fork** repositori ini ke akun Anda.
2.  **Clone** *fork* Anda ke komputer lokal:
    ```bash
    git clone [https://github.com/](https://github.com/)[Username-Anda]/Maritime-Artificial-Intelligence-Internet.git
    ```
3.  Buat **Branch Baru** untuk perubahan Anda:
    ```bash
    git checkout -b fitur/koreksi-typo-minggu-01
    ```
    (Gunakan nama *branch* yang deskriptif, misal: `bug/perbaiki-plot-minggu-02` atau `ref/tambah-paper-sonar`)
4.  **Lakukan Perubahan:** Edit file, tambahkan kode, atau perbaiki *typo* Anda.
5.  **Commit** perubahan Anda dengan pesan yang jelas:
    ```bash
    git add .
    git commit -m "docs: Perbaiki typo di materi Minggu 01"
    ```
6.  **Push** ke *branch* Anda di *fork* Anda:
    ```bash
    git push origin fitur/koreksi-typo-minggu-01
    ```
7.  Buka **Pull Request (PR)** dari *branch* Anda di *fork* Anda ke *branch* `main` repositori ini.

Saya akan meninjau PR Anda sesegera mungkin. Terima kasih!
