## ğŸš€ Quick Start

Panduan ini untuk menjalankan kode-kode praktikum dan latihan di dalam repositori ini.

### Prasyarat

* Python 3.9+
* Anaconda atau `pip` package manager

### Instalasi

1.  **Clone repositori ini**
    ```bash
    git clone [https://github.com/](https://github.com/)[Username-Anda]/[Nama-Repo-Anda].git
    cd [Nama-Repo-Anda]
    ```

2.  **Buat virtual environment (direkomendasikan)**
    ```bash
    python -m venv venv
    source venv/bin/activate  # Di Windows: venv\Scripts\activate
    ```

3.  **Install dependencies**
    ```bash
    pip install -r requirements.txt
    ```

4.  **Jalankan Jupyter Notebook**
    ```bash
    jupyter notebook
    ```
    Lalu, navigasi ke folder `Minggu-XX` yang ingin Anda lihat.

## ğŸ“š Struktur Pembelajaran 16 Minggu

Setiap folder `Minggu-XX` berisi 9 komponen inti:

1.  **Materi:** Catatan dan rangkuman teori.
2.  **Praktik:** Laporan praktikum terpandu (Lab).
3.  **Latihan Mandiri:** Solusi untuk tugas mandiri.
