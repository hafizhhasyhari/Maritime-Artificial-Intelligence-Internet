UTS adalah proyek dasbor web interaktif. Ini menguji kemampuan kita mengintegrasikan semua yang telah dipelajari.

## Checklist Topik

* [ ] **Minggu 1: Fondasi Data**
    * [ ] Tahu bedanya data tabular, spasial, temporal.
    * [ ] Bisa baca CSV/Excel pakai `Pandas`.
    * [ ] Bisa ambil data dari API (pakai `requests`).
* [ ] **Minggu 2: Web GIS**
    * [ ] Paham beda Raster vs. Vektor.
    * [ ] Paham apa itu GeoJSON dan CRS.
    * [ ] Bisa buat peta Folium sederhana (tambah *marker*, *popup*).
* [ ] **Minggu 3: Klasifikasi ML**
    * [ ] Paham alur kerja `sklearn` (fit, transform, predict).
    * [ ] Bisa jelaskan `train_test_split` dan tujuannya.
    * [ ] Tahu cara menghitung Akurasi.
* [ ] **Minggu 4: Analisis Spasial**
    * [ ] Bisa baca file Shapefile/GeoJSON pakai `Geopandas`.
    * [ ] Bisa melakukan *query* spasial (misal: `gpd.sjoin`).
    * [ ] Bisa membuat *heatmap* (peta panas).
* [ ] **Minggu 5: Visi Komputer (CV)**
    * [ ] Paham konsep dasar CNN (Convolutional Neural Network).
    * [ ] Tahu cara me-load model yang sudah dilatih (misal: `.h5`).
    * [ ] Bisa melakukan *image classification* sederhana.
* [ ] **Minggu 6: AI Generatif**
    * [ ] Tahu beda model Diskriminatif vs. Generatif.
    * [ ] Paham cara kerja *prompt engineering* sederhana.
* [ ] **Minggu 7: NLP (Arsip Sejarah)**
    * [ ] Paham apa itu *preprocessing* teks (stopwords, stemming).
    * [ ] Tahu konsep TF-IDF.
    * [ ] Tahu konsep *Topic Modeling* (LDA).

## Fokus Proyek UTS

* Dasbor harus interaktif (bisa diklik).
* Harus ada minimal 1 elemen Peta (Folium/Geopandas).
* Harus ada minimal 1 elemen AI (misal: input data -> keluar prediksi ML, atau input gambar -> keluar klasifikasi CV).
* (Opsional) Tampilkan hasil Generatif AI atau analisis NLP.
